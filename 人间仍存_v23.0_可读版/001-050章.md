- **核心矛盾**：整条街因此难以突破，但街内伤员也成了承重结构的一部分。
- **本章回报 / 后果**：小满找到唯一未被接入总图的排水沟，导淮从地下撤人。
- **超凡 / 战斗表现**：韩崇岳脊背浮出横梁纹，街屋同时发出受压声；断开并梁后，所有累积损伤重新落下。

### 第045章《旧渠还能走》
- **主驱动人物**：陈亦、田水嫂
- **核心事件**：三派争论泥足道路时，突发溃水逼众人进入泥足当年留下的旧渠。
- **核心矛盾**：真实历史没有错，后人的维护与解释才冲突。
- **本章回报 / 后果**：众人亲身确认泥足工程有效，争论转向谁有资格继承。
- **战斗 / 超凡表现**：陈亦以淤泥不吞和旧渠灾害降低护住塌渠，裴玄纪短暂压住水势。

### 第046章《第七井挖到没有尽头的灰黑》
- **主驱动人物**：旧河工群像
- **核心事件**：多年前河工挖出埋在旧河心的巨型楔体，露出部分如地下石殿，边缘未尽。
- **核心矛盾**：它不能搬走，正常工具也无法切开。
- **本章回报 / 后果**：大楔体量成立。

### 第047章《世界上不止这一块楔》
- **主驱动人物**：陆执灯旧档
- **核心事件**：陆档记载其他地域也可能发现盲目织史者残骸凝成的小楔或异形楔。
- **核心矛盾**：不能把大改淮巨楔写成世界唯一。
- **本章回报 / 后果**：世界尺度拓展。

### 第048章《沈怀保住的是发现者名单》
- **主驱动人物**：沈怀旧痕
- **核心事件**：沈怀记录谁先挖到、谁先触碰、谁随后消失。
- **核心矛盾**：发现者逐步被从各类记录中抹去。
- **本章回报 / 后果**：沈怀卷入理由成立。

### 第049章《楔只认最后刻下的结果》
- **主驱动人物**：杜衡、陆执灯
- **核心事件**：所有前置工程、文书、印信、证词都不能直接决定结算。
- **核心矛盾**：它们唯一能影响的是能力大方向。
- **本章回报 / 后果**：核心机制钉死。

### 第050章《沈怀的咳嗽》
- **主驱动人物**：曹留、陈亦
- **核心事件**：曹留取出一片保存沈怀旧声的铜片，让陈亦听见原身生前一次极短的咳嗽和半句抱怨。
- **核心矛盾**：声音熟悉，陈亦却没有任何记忆回应。
- **本章回报 / 后果**：曹留确认身体位置延续，声口与人格已经不同。
- **超凡表现**：残声在屋里绕了一圈，最后停在陈亦肩后；墨未干看见声音与身份之间仍有未落定的缝。
