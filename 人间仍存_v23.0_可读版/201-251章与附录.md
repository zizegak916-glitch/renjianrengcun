- 核心物：旧实灯

他查假账、找真实、记录真实。

他与陈亦之间始终同时存在保护、利用、观察和警惕。

讹视让他看见极淡格式错位，过度使用会混淆当前版本与旧版本。

撑缝不是停时，也不是改写结果。它让本该闭合的动作暂时接不上：

- 笔尖落不下；
- 印泥粘住；
- 拔刀第二寸接不上；
- 伤口闭不上；
- 结论卡在出口。

他不会沿着“只保记录到永远先救人”的直线成长。他可能在一处先救人，也可能在另一处坚持保住材料。

## 杜衡
只信熟悉流程刀口的人能活 → 发现陈亦这种流程外异常也能滚出缝 → 留入口但不收编。

## 老孟
只认旧人旧物旧样子 → 承认陈亦的不同选择也能接住一段人间关系 → 面对导淮内部欲望，不再把“我们是一家人”当答案。

## 罗小满
- 姓名：罗小满
- 称谓：跑线人
- 年轻、嘴快、跑线快、敢冒险
- 对路径、撤离口令与现场方向高度敏感
- 常被导淮老人护着

她的强处不是正面轰杀，而是识路、改路、传口令、借地形，并在追逐中把人导向活路。

导引失控时，她会把自己人也带入错误路径。

她不会因一次错误突然成熟。她可能这次压住最急的话，下次又因熟人受伤提前开口；会顶撞孟从周，也会在真正害怕时听话。

## 裴玄纪
总图能代表整体 → 总图不断删掉具体人 → 在终局承认总图并非一个人完成，但仍坚持需要整体判断。

## 沈确
拆分可以防止独占 → 拆到无人完整负责 → 终局保留一条不能再拆的责任。

## 顾廷璋
只有超越个人的皇权空位能收束争夺 → 空位无法替活人握刀 → 放弃让旧印直接承接，但不放弃对末帝的信仰。

## 柳听春
用轻浮拒绝任何大义绑架 → 发现有些事确实需要有人站一下 → 用自己最擅长的场面与传言保护撤离，而非突然变严肃。

## 白砚奴
一切风险都可称量 → 大改淮最终承接无法在落定前估值 → 首次承认不可称量，但仍把陈亦视为长期异常资产。

## 第七副眼
观察者可以不入局 → 观察报告改变三派路线 → 在全神州星屑中被迫选择救人或保样本。

## 曹留
只认沈怀旧声 → 死人残留无法替活人走下一步 → 在陈亦做出不同选择后仍帮他截声。

## 谢观成
边界必须由某人划下 → 每条线都把具体人留在另一侧 → 第一卷只迈一步，主动站到自己划出的危险一侧。

## 韩照渠
结构不塌就有余地 → 撑住也可能把人压成结构的一部分 → 水牢中改变支撑点，第一次把人放在结构前。

## 梁肃川
看见最后动作能盖过前面所有人 → 相信最后活着并落刀的人理应拿走一切 → 被陈亦从“抢刀”转成“改变事件”击破，但仍获得残缺结果并存活。

## 周敬山
为救家人出卖一次路线 → 发现背叛也能换来自己的安全 → 第二次主动出卖，最终被利用并灭口。

## 段成业
想让导淮拥有自己的力量 → 把“共有力量”逐渐偷换成“由我先拿到” → 计划失败，但提出的权力问题不会消失。

# 附录二｜跨卷保留钩子

- 末帝：原稳定未来中有沈怀位置，没有陈亦人格。现世位于碎卷海，新能力无普通代价。
- 断流荒：收藏态，除外来者无活物继续产生新变化。
- 其他楔：世界各处可能存在不同规模残骸，异域楔刀证明物线已跨地域流通。
- 导淮：组织去中心化，但第三份间谍名单证明渗透未尽。
- 奉玺派：顾廷璋放弃旧印直接承接，不等于放弃末帝与皇权空位。
- 极北：第七副眼的观察已被陈亦污染，后续将产生体系反应。
- 银号：白砚奴无法给大改淮估值，却对陈亦保留长期称量兴趣。
