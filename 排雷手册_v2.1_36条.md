## ▎26条硬约束

### 禁止类（命中即 red flag）

1. ❌ **"死过一回后变了"** — 不准用任何人物的旁白/台词解释陈亦"变了"。陈亦就是现场接招，不解释身份。
2. ❌ **"他不一样了"** — 同上，所有替陈亦圆场的文字一律删。
3. ❌ **"不记得了"/"记不清了"/"死过忘事"** — 陈亦不是失忆，是没有沈怀记忆。对话中对旧关系的回应必须用反问/转利益判断，不能含糊。
4. ❌ **"沈怀的记忆"出现在叙述层** — 陈亦零沈怀记忆。身体有肌肉记忆（握笔、抄录），但脑子没有画面、名字、事件。
5. ❌ **旧身位预排/旧习惯自动生效** — 刺客按沈怀旧身位埋伏，陈亦应该因为不懂旧身位躲偏。
6. ❌ **角色替陈亦遮挡身份** — 任何角色说"他死过所以不一样了"=替主角圆场，删。
7. ❌ **"候"作为章内名词出现** — 候只放在组说明和后台备注区，正文中不出物理/台词实体。
8. ❌ **"补痕"作为人物对话中的解释性名词** — 补痕是叙述层描写，不能成为角色讨论的东西。
9. ❌ **元叙事/文学总结** — "他感到""读者可能""这一天他明白了"。写了直接删段。
10. ❌ **"本章代价""本章风险"** — 大纲字段，不准进入正文。
11. ❌ **第一人称 "我" 做内心独白** — 全文第三人称，用"陈亦"或"他"。
12. ❌ **破折号 >5/章** — `grep -c '——'` 检查，超过必须修改。
13. ❌ **候源/活人锚点/冷损/胸冷/体弱/写没/候而未决** — 全卷零命中。
14. ❌ **诗意比喻/情绪形容词** — "像一条命运的河流"、"他感到深深的悲伤"。
15. ❌ **角色说出"候到了""候来了"** — 候不出现在台词层。

### 身份线正确约束

16. ✅ **面子上陈亦就是沈怀** — 所有人按沈怀对待他：抛旧话、旧债、旧习惯、旧怨气。
17. ✅ **陈亦现场接招** — 不知道答案时：观察→反问→抓关键利益点→当场转"谁要写死我"。
18. ✅ **不准解释身份** — 没有"我不是沈怀"、"他死过一回"、"他变了"，只有对现场的回答。
19. ✅ **暗语接不上 → 反问'接错一句你就写我回死的？'** — 这不是回暗语，这是抓对方也怕的的点。
20. ✅ **旧物/旧话抛过来 → 判断'能救人的还是能害我的'** — 陈亦不回忆旧物意义，只看现在能用来做什么。

### 写作质量约束

21. ✅ **每段1-3句** — 不要长段落堆砌。
22. ✅ **乐子人底色** — 陈亦反应词："有意思""行吧""来都来了"。70%+ 内心独白是乐子人风格。
23. ✅ **笑点是主导** — 不是写完严肃再加笑点，是用乐子人视角写一切。每章至少3-5处乐子人内心吐槽。
24. ✅ **人间链是鸡毛蒜皮** — 欠两碗面、烫伤认手、赈粥掺水、门口晾的湿棉裤。不是宏大叙事链。
25. ✅ **日常嵌进危机** — 日常（吃饭、欠钱、字丑）只能嵌进危机动作，不能打断危机。
26. ✅ **字数 2000-3300** — 批量写 2200-2500 字。超过检查是否有冗余环境描写。

### 前期认知负荷控制（第1-6章）

27. ✅ **前6章陈亦受限视角** — 不做体系讲解，不解释世界观机制。死销、复销、补痕、未定材料等术语可以出现，但只通过纸面、动作、旁人反应让读者感知，不做定义性叙述。
28. ✅ **陈亦前3章认知边界** — 只理解四件事：有人要按我手、纸上写我死了、落款人怕担责、我不按就还有活路。超出这四件事的认知一律不写进陈亦的内心/叙述。
29. ✅ **禁止前6章做体系讲解** — 世界观、能力机制、派系结构、历史背景一律不进入前6章正文。全部延后到第7章起渐次展开。

### 补痕/候边界强化（第67章、第173章）

30. ✅ **第67章：补痕边界** — 用剧情动作表现补痕只能顺已有痕迹往前牵，不能补愿望、不能补完整真相、不能把过去完整显影。让陈亦补不到完整敲闸声，以此展示补痕的上限。
31. ✅ **第173章：候边界** — 让小物回到具体人手里，形成新事件。候来自陈亦/沈怀真实进入未结完事件后继续留下新事件，并被旧日本质接受后随机结算；不是补旧痕、不是补愿望，也不是单独“新行动被记录”。
32. ✅ **补痕禁止补愿望** — 补痕只能顺已有痕迹往前牵，绝不能写成"陈亦想怎样就补出什么"。

### 沈怀残留危险化

33. ✅ **沈怀残留形式** — 不写成夺舍或灵魂说话。只写成：身体旧反应（握笔姿势、走路步幅）、旁人记忆压力（旧识用沈怀标准要求陈亦）、旧习惯影响判断（沈怀的惯性反应把陈亦带偏）。
34. ✅ **沈怀残留双刃** — 必须双刃：有时救陈亦一寸（旧身体反应躲过致命一击），有时把他推回沈怀旧死位（旧习惯让他做出来沈怀才会做的事、暴露破绽）。
35. ✅ **沈怀残留埋设章节** — 第13章、第66章、第84章、第88章埋线；卷末让"沈怀旧称呼压回陈亦身上"写得更具体（不是概念性的"被当成沈怀"，而是具体某个旧称呼、旧身份标签重新捆住他）。

### 账面术语控制

36. ✅ **账面术语 ≤10 个** — 全书账面术语控制在10个以内。介于账本质和账只是神州称呼，减少账面术语，增加白话类称呼（如"记你的名字""写上""画掉""不认"等）。新增术语必须经用户确认。

## 审查流程

每章交付前执行：

```bash

wc -m chapter.txt

# 2. 禁用词（含沈怀残留形式禁词）
grep -nE '冷损|胸冷|体弱|写没|候源|活人锚点|本章代价|候而未决|第一人称|不是沈怀|死过一回|变了|不记得|记不清|夺舍|灵魂|他在心里说|他心想' chapter.txt

# 3. 破折号

# 4. 元叙事
grep -nE '我感到|他感到|读者可能|读者会|明白了|明白了什么|体系|机制|世界观' chapter.txt

grep -nE '补出|补到.*会|补.*希望|补.*想要|补.*未来' chapter.txt

# 6. 账面术语计数（手动核查，目标≤10个）
```
